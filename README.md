# PythonRayCasting2D
Visualization of Ray Casting technology in python.
### How to change map?
Map example:
```python
text_map = [
    'WWWWWWWWWWWW',
    'W.WW.......W',
    'W......W...W',
    'W..........W',
    'W..........W',
    'W....WWWW..W',
    'WW.........W',
    'WWWWWWWWWWWW'
]
```
+ Open a **map.py** file
+ Find the **"text_map"** variable
+ Change map

**Note:** _The wall is indicated by the symbol ***'W'***, and empty space by the symbol ***'.'***_
